# 3.3.3 변경 내역

- PDF 이미지 OCR에 한글 ML Kit, 영문 ML Kit, Tesseract `tessdata_fast` 한/영 보조 엔진을 결합해 한글·영문 혼용 페이지의 검색 색인 품질을 개선
- 현재 보고 있는 PDF 페이지는 일정 시간 머문 뒤 품질 모드로 OCR하고, 주변·나머지 페이지는 스크롤/필기 입력이 멈춘 유휴 상태에서만 처리하도록 변경
- 손글씨 자동 OCR 색인은 한국어 고정 인식 대신 `ko`와 `en-US` 후보를 줄 단위로 비교하는 자동 모드로 변경
- 대용량 PDF에서는 이미지 OCR 입력 크기를 줄이고 페이지 사이 처리 간격을 늘려 저사양 모바일 부하를 완화
- 페이지 전환 이벤트를 추가해 OCR 체류 타이머가 실제 현재 페이지 기준으로 초기화되도록 수정
