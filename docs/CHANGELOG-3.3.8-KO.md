# 3.3.8 변경 내역

- 대용량 문서에서 페이지 렌더 요청을 프레임 큐로 나눠 스크롤 중 멈춤과 튕김 위험을 줄였습니다.
- 화면 밖 캔버스는 backing store를 1x1로 줄여 GPU/메모리를 즉시 반환하도록 수정했습니다.
- PDF 배경 이미지와 삽입 이미지 캐시를 대용량 문서에서는 더 작게 유지해 메모리 압박을 낮췄습니다.
- 자동 OCR의 화면 페이지 계산이 모든 페이지 DOM을 순회하지 않고 앱의 페이지 간격 계산을 사용하도록 최적화했습니다.
- 520페이지 스트레스 벤치마크 기준 최대 스크롤 시간이 1.4초대에서 567ms 이하로 줄었습니다.
