# 3.3.10 변경 내역

- 확대 상태에서 페이지가 화면보다 넓어질 때 좌우 스크롤 가능한 폭을 확보했습니다.
- 손 도구 드래그, 확대 상태 터치 팬, 트랙패드/Shift 휠로 좌우 이동할 수 있도록 보강했습니다.
- PDF·이미지 배경 페이지를 200~300% 확대할 때 캔버스 backing store를 약 700만 픽셀 이하로 제한하고 고배율 마운트 캔버스 수와 이미지 캐시를 줄였습니다.
- 자동 도형 변환에서 화살표 판정을 제거했습니다.
- 흔들린 사각형도 사각형으로 더 잘 잡도록 경계선 기반 사각형 점수를 추가했습니다.
- 오각형·육각형은 변 길이, 각도, 중심 반경이 충분히 맞는 경우에만 자동 변환하도록 과인식을 줄였습니다.
- 원·정사각형·삼각형·오각형·육각형 등 정형 도형은 생성/조정 초기에 비율을 잠그고, 사용자가 충분히 비율을 벗어나게 드래그하면 잠금을 해제합니다.
