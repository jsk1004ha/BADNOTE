# 3.3.11 변경 내역

- 도형 도구에 수동 곡선 그리기를 추가했습니다.
- 곡선 도형은 드래그 방향에 맞춰 제어점을 자동 보정해 처음부터 곡선으로 보이게 했습니다.
- 라이브러리 상단과 폴더 영역에 새 폴더 생성 기능을 추가했습니다.
- 생성한 폴더는 기기에 저장되고, 같은 이름을 다시 만들면 기존 폴더로 이동합니다.
- 작은 범위에서 국소적으로 문지른 낙서도 지우기 제스처로 인식되도록 별도 밀도 판정을 추가했습니다.
- S Pen 버튼이 입력 도중 감지되면 이미 시작된 펜 획을 버리고 즉시 지우개 세션으로 전환합니다.
- S Pen 버튼을 닿기 전 누른 경우와 닿은 뒤 누른 경우를 모두 회귀 테스트에 추가했습니다.
