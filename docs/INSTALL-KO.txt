bad note 3.3.21 설치 안내
=====================================

권장 APK
--------
1) 업데이트용: bad-note-Android-3.3.21-Update.apk
   패키지 ID: com.inkforge.note4

2) 병행 설치용: bad-note-Android-3.3.21-SideBySide.apk
   패키지 ID: com.inkforge.note5

공통 정보
---------
버전 코드: 351
버전 이름: 3.3.21
최소 Android: API 23
대상 Android: API 36
지원 ABI: arm64-v8a, armeabi-v7a

3.2.0 사용자에게 중요한 사항
---------------------------
이 소스 패키지에는 기존 배포 서명키가 포함되어 있지 않습니다. 3.3.21 업데이트용 APK는 패키지 ID는 같지만 로컬 임시 키로 서명했으므로 Android가 기존 앱 위의 직접 업데이트 설치를 거부할 수 있습니다.

1. 3.2.0에서 중요한 노트를 .ifnote 파일로 내보냅니다.
2. 백업 파일이 실제로 저장됐는지 확인합니다.
3. 기존 com.inkforge.note4 앱을 삭제합니다.
4. 3.3.21 병행 설치용 APK를 설치합니다.
5. 3.3.21에서 .ifnote 파일을 가져옵니다.

기존 앱을 삭제하지 않으려면 병행 설치용 APK를 사용하십시오. 병행 설치판은 별도 앱 저장 공간을 사용하므로 기존 노트가 자동으로 나타나지는 않습니다.

첫 실행
-------
- 한글 필기, 숫자·기호, 도형용 Digital Ink 모델을 내려받기 위해 인터넷 연결이 필요할 수 있습니다.
- 모델 다운로드가 끝난 뒤 필기 인식은 기기 내에서 실행됩니다.
- 번들형 한글·영문 이미지 OCR은 APK 안에서 동작합니다. Tesseract `tessdata_fast` 한/영 모델은 보조용으로 포함되어 있지만 자동 OCR 기본 경로에서는 사용하지 않습니다.

설치 오류
---------
- “앱이 설치되지 않음”: 동일 패키지의 다른 서명 앱이 있으면 백업 후 기존 앱을 제거합니다.
- “알 수 없는 앱 설치 차단”: Android 설정에서 다운로드에 사용한 앱의 설치 허용을 켭니다.
- Android 6 이전 기기는 지원하지 않습니다.
