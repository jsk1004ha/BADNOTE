# 3.3.4 변경 내역

- 편집기 오른쪽에 페이지 스크롤 바와 페이지 번호 이동 입력을 추가
- 화면에 보이는 필기 페이지를 기준으로 자동 OCR 색인을 예약하고, 사용자가 머문 뒤 유휴 상태에서 처리하도록 조정
- PDF 이미지 OCR 자동 경로에서 Tesseract 사용을 끄고 ML Kit 한글·영문 OCR을 우선 사용
- OCR 준비 상태 알림은 처음만 잠시 표시한 뒤 사라지도록 변경
- S Pen 측면 버튼을 누르는 동안 지우개로 동작하고, 놓으면 이전 도구로 복귀하도록 수정
- 손글씨 수식 자동 계산 기본값을 끄고, 문서 옵션에서 현재 페이지 수식을 수동 계산할 수 있도록 추가
- 수식 계산 결과는 식과 결과를 함께 표시하는 손글씨 옆 계산 객체로 삽입
- `\frac{}`, `\sqrt{}` 등 일부 LaTeX 형태를 로컬 계산식으로 정규화
- 도형 자동 변환의 유지 시간을 늘리고, 삼각형·사각형·원/타원 판정을 보강
- 지그재그 지우기가 도형 변환과 충돌하지 않도록 조건을 분리하고, 도형 지우개 hit-test를 외곽선 중심으로 개선
- 확대 렌더링 시 캔버스 해상도를 표시 크기에 맞춰 다시 잡아 필기 선이 흐려지지 않도록 개선
- 두 손가락 핀치 확대가 페이지 넘김으로 잘못 해석되는 문제를 수정
