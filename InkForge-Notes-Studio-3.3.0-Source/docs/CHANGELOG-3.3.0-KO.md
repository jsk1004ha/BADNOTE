# 3.3.0 변경 내역

- 한글 필기 OCR을 Android ML Kit Digital Ink `ko` 모델로 교체
- OCR 줄 분리, 필기 영역·앞 문맥 전달, 자동 검색 색인 추가
- PDF 스캔 페이지용 번들형 한국어 이미지 OCR 추가
- 손글씨 수식 후보 병합·기호 정규화·자동 계산 개선
- 새 노트 생성 단계의 PDF 가져오기 추가
- PDF 페이지를 Blob 자산으로 분리하고 6페이지 단위 증분 저장
- 연속 문서 페이지 캔버스 가상화 및 썸네일 지연 렌더링
- 그리기 프레임 합치기, 페이지 단위 실행 취소, 긴 문서 자동 저장 간격 최적화
- 직선·곡선·원·타원·사각형·다각형 자동 변환 개선
- 네이티브 기본 도형 분류기 보조 경로 추가
- Android 스타일러스 압력·기울기·방향·호버·지우개 촉·측면 버튼 이벤트 연결
- 마지막 페이지에서 당겨 새 페이지 추가
