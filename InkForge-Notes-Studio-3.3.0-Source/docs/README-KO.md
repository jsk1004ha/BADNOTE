# bad note 3.3.11 기능 안내

## 3.2.0 기반 유지

기존 라이브러리, 폴더·즐겨찾기·휴지통, 아이콘 중심 편집 UI, 8종 펜, 형광펜, 지우개, 올가미, 텍스트, 이미지, 오디오, 스티키 노트, 암기 가림 테이프, 색상표·색상 조합기, 검색, 실행 취소·다시 실행, 페이지 추가·복제·이동·삭제, 북마크, 확대·축소, PNG·PDF·IFNOTE 내보내기를 유지합니다.

## 라이브러리 폴더

- 라이브러리 상단의 폴더 생성 버튼 또는 폴더 영역의 **새 폴더** 칩으로 폴더를 만들 수 있습니다.
- 만든 폴더는 기기 저장소에 보존되며, 폴더 안에서 새 노트를 만들면 해당 폴더에 바로 들어갑니다.
- 같은 이름의 폴더를 다시 만들면 중복 생성 대신 기존 폴더로 이동합니다.

## 한글·영문 손글씨 OCR

- 일반 필기 획은 Android 네이티브 ML Kit Digital Ink의 `ko`와 `en-US` 후보를 줄 단위로 비교해 한글·영문 혼용 필기를 인식합니다.
- 여러 줄 필기를 줄 단위로 분리하고, 필기 영역과 앞 문맥을 인식기에 전달합니다.
- 화면에 보이는 페이지에 잠시 머문 뒤 유휴 상태가 되면 숨김 `ocrIndex` 객체를 갱신해 문서 검색에 등록합니다.
- PDF에 텍스트 레이어가 없으면 ML Kit 한글·영문 이미지 OCR로 페이지를 색인합니다. Tesseract 보조 경로는 앱에 남아 있지만 자동 OCR 기본값에서는 사용하지 않습니다.
- 모델 다운로드 또는 네이티브 인식이 불가능한 환경에서는 기존 로컬 인식기로 대체합니다.

설정의 **손글씨 OCR 자동 등록**에서 자동 색인을 끌 수 있습니다.

## 손글씨 수식 계산

손글씨 수식 자동 계산은 기본값이 꺼져 있습니다. 문서 옵션의 **손글씨 수식 계산**을 누르면 현재 페이지의 최근 필기 수식을 인식하고, 수식과 결과를 손글씨 옆의 계산 객체로 함께 표시합니다.

지원 범위: 사칙연산, 괄호, 소수, 백분율, 거듭제곱, 제곱근, `sin`, `cos`, `tan`, `log`, `ln`, `abs`, `pi`, `e`.

일부 LaTeX식 `\frac{}`, `\sqrt{}` 형태는 로컬 파서가 계산식으로 정규화합니다. 복잡한 적분, 행렬, 연립방정식, 다단 분수 전체를 해석하는 서버형 HMER/CAS 파이프라인은 포함하지 않았습니다.

## PDF 새 노트와 대용량 최적화

- 새 노트 메뉴의 **PDF로 새 노트**에서 PDF를 곧바로 문서로 만듭니다.
- 기존 문서에 PDF를 삽입하는 경로도 유지합니다.
- PDF 페이지는 문서 JSON 안에 base64로 중복 저장하지 않고 IndexedDB의 Blob 자산으로 분리합니다.
- PDF 텍스트 레이어를 추출해 검색에 즉시 포함합니다.
- 페이지를 한 번에 모두 화면에 그리지 않고 현재 화면 부근 캔버스만 마운트합니다.
- 사이드바 썸네일도 보이는 항목만 지연 렌더링합니다.
- 입력 중 그리기는 `requestAnimationFrame` 단위로 합쳐 처리하고, 긴 문서는 저장 간격과 실행 취소 메모리를 조절합니다.
- 확대 시 캔버스 backing store를 현재 표시 크기에 맞춰 다시 렌더링해 필기 선이 흐려지지 않도록 했습니다.
- 확대 상태에서도 낙서 지우기와 도형 자동 변환은 화면 픽셀 기준으로 판정해 확대율 때문에 다른 페이지 좌표처럼 해석되는 문제를 줄였습니다.
- 확대 상태에서 손 도구 드래그, 터치 팬, 트랙패드/Shift 휠로 좌우 이동할 수 있습니다.
- PDF·이미지 배경 페이지는 200~300% 확대 시 캔버스 픽셀 예산과 이미지 캐시 수를 제한해 메모리 폭증을 줄입니다.

## 도형 자동 변환

펜으로 도형을 그리고 끝에서 잠시 멈추면 다음을 자동 변환합니다.

- 직선, 화살표
- 자유 곡선
- 원, 타원
- 정사각형, 직사각형
- 삼각형, 오각형, 육각형

Android에서는 ML Kit 기본 도형 분류기 결과도 보조로 사용합니다. 도형 하나를 한 번에 그리고 마지막 지점에서 잠시 유지할 때 가장 안정적입니다.

낙서 지우기처럼 짧고 촘촘한 지그재그 입력은 도형 변환보다 먼저 지우기 제스처로 처리합니다. 작은 범위에서 국소적으로 문질러도 왕복 방향 전환과 선 밀도가 충분하면 지우기로 인식합니다.

자동 변환에서 화살표 판정은 제외했습니다. 사각형은 흔들린 모서리도 더 적극적으로 인식하고, 오각형·육각형은 정교하게 그린 경우에만 자동 변환합니다.

도형 도구에는 수동 **곡선** 버튼이 추가되어, 드래그만 해도 제어점이 자동으로 잡힌 곡선을 만들 수 있습니다.

## S Pen·Android 스타일러스

네이티브 `MotionEvent` 경로로 압력, 기울기, 방향, 호버 거리, 펜촉/지우개 촉, 측면 버튼 상태와 스타일러스 키 이벤트를 수집합니다. 화면에 닿기 전 또는 닿은 상태에서 S Pen 버튼을 누르면 누르는 동안 지우개로 동작하고, 놓으면 이전 도구로 돌아갑니다.

- 지우개 촉: 접촉 중 지우개 전환
- 측면 버튼: 입력 도중 감지되어도 진행 중인 펜 획을 버리고 즉시 지우개 세션으로 전환
- 스타일러스 키 이벤트: 지원 기기에서 실행 취소/다시 실행 또는 페이지 이동

기종·펌웨어가 일반 Android 스타일러스 이벤트로 전달하지 않는 Bluetooth Air Actions는 별도 Samsung Remote SDK 연동이 필요합니다.

## 페이지 추가

연속 페이지 보기에서 마지막 페이지 맨 아래까지 이동한 뒤 위로 더 당기고 놓으면 새 페이지가 추가됩니다. 기존의 페이지 하단 추가 버튼도 유지됩니다.
