# 3.3.2 변경 내역

- 자동 OCR 설정 저장 키를 통일해 재실행 후에도 손글씨 OCR 자동 등록 설정이 유지되도록 수정
- 새 노트의 PDF 가져오기 버튼을 메인 액션 라우터에도 직접 연결
- 대용량 PDF의 이미지 OCR을 현재 페이지 주변부터 지연 처리하도록 변경해 가져오기 직후 부하를 완화
- Android 네이티브 스타일러스 이벤트를 WebView 포인터 처리와 연결해 S Pen/지우개 판정을 보강
- 수식 필기 인식에서 `+`, `-`, `=`, `*`, `/` 같은 기본 기호 후보를 기하 휴리스틱으로 보강
- 명확한 직선, 폐곡선, 곡선의 자동 도형 변환 민감도를 개선
- Windows build-tools의 `apksigner.bat`, `zipalign.exe`, `aapt2.exe`를 검증 스크립트가 인식하도록 수정
