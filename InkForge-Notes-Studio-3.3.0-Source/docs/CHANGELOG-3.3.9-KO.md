# 3.3.9 변경 내역

- 확대 상태에서 낙서 지우기와 도형 자동 변환의 제스처 판정을 화면 픽셀 기준으로 분리했습니다.
- 실제 저장되는 필기 획과 도형 좌표는 문서 기준으로 유지해 확대·축소 후에도 렌더링 품질이 유지됩니다.
- 짧고 촘촘한 지그재그 입력은 도형 변환보다 낙서 지우기 제스처로 먼저 처리하도록 우선순위를 조정했습니다.
- 3.8배 확대 상태에서 낙서 지우기가 기존 획을 지우고 새 도형을 만들지 않는 회귀 테스트를 추가했습니다.
- 확대 직후 드래그 잠금 테스트가 느린 헤드리스 렌더 환경에서도 기능 동작을 올바르게 검증하도록 조정했습니다.
