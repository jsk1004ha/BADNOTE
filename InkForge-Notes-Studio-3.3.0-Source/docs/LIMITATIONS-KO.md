# 3.3.0 검증 범위와 제약

## 검증한 항목

- JavaScript 구문 검사
- 브라우저 자동 회귀 시험: UI 겹침, 페이지 추가, 확대, 자동 수식 계산, OCR 검색 색인, 색상 조합기, 불투명 가림 테이프, 기본 도형 인식, S Pen 제스처 경로
- 3페이지 PDF를 새 노트로 가져오기, PDF 텍스트 추출, Blob 자산 저장
- 150페이지 문서에서 페이지 래퍼 150개 중 근접 캔버스 3개만 마운트하는 가상화 시험
- Android Java 컴파일과 두 릴리스 변형 Gradle 빌드
- APK ZIP 무결성, 16 KiB 페이지 정렬, v1/v2/v3 서명 검증

## 검증하지 못한 항목

이 빌드 환경에는 Galaxy 태블릿, S Pen과 Android 물리 기기가 없으므로 실제 설치 버튼, 장시간 필기 지연, 펜 기종별 압력 곡선, 제조사별 Air Actions는 실기기에서 측정하지 못했습니다.

## 인식 제약

- 손글씨 인식 정확도는 글씨체, 획 순서, 글자 간격과 필기 영역에 영향을 받습니다.
- 첫 Digital Ink 모델 다운로드가 끝나기 전에는 로컬 대체 인식기를 사용할 수 있습니다.
- 수식 기능은 범용 필기 인식 후보와 내장 계산 파서를 조합합니다. 적분·행렬·복잡한 분수 구조를 인식하는 전용 CAS/수학 OCR은 아닙니다.
- 도형 자동 변환은 도형 하나를 연속해서 그리고 끝에서 잠시 유지할 때 가장 안정적입니다.

## 대용량 문서

화면 렌더링과 메모리 사용을 줄였지만 PDF 전체 페이지를 가져오고 압축 자산으로 만드는 시간과 저장 용량은 페이지 수·해상도에 비례합니다. 저장 공간이 부족하면 가져오기가 실패할 수 있습니다.

## 서명

소스 패키지에는 개인 릴리스 키가 포함되지 않습니다. 제공 APK와 동일한 서명으로 후속 업데이트를 배포하려면 별도로 안전하게 보관한 배포 키가 필요합니다. 3.2.0의 이전 임시 키는 복구할 수 없어 직접 덮어쓰기 업데이트는 지원하지 않습니다.
