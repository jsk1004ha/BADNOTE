# 3.3.18 변경 내역

## 라이브러리 중첩 폴더

- 폴더 안에서 다시 새 폴더를 만들 수 있도록 폴더 저장 데이터에 `parentId`를 추가했습니다.
- 기존 폴더 데이터는 자동으로 문서 루트 하위 폴더로 보정됩니다.
- 현재 폴더의 하위 폴더만 폴더 영역에 표시하고, 상단 breadcrumb는 여러 단계의 경로를 표시합니다.
- 같은 부모 폴더 안에서만 폴더 이름 중복을 막아 서로 다른 폴더 안에는 같은 이름의 하위 폴더를 만들 수 있습니다.
- 폴더 삭제 시 직접 들어 있던 노트와 하위 폴더는 삭제하지 않고 상위 폴더로 이동합니다.
- 하위 폴더 생성, 하위 폴더 안 노트 생성, 하위 폴더 삭제 후 노트 이동까지 웹 회귀 테스트에 추가했습니다.
