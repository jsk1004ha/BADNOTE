# 3.3.16 변경 내역

## 입력 안정화

- 손 잠금 상태에서 터치 입력이 최근 S Pen 네이티브 이벤트 때문에 펜 입력으로 오인되어 새 획이 생기는 문제를 막았습니다.
- 쓰기 도구 사용 중 비펜 입력은 새 필기 세션을 시작하지 않도록 차단하고, 손 도구의 이동 동작은 유지했습니다.

## 문질러 지우기

- 문질러 지우기의 실제 삭제 반경을 화면 픽셀 기준으로 더 좁혀, 문지른 경로에 닿지 않은 주변 객체가 같이 삭제되는 문제를 줄였습니다.
- 가까운 평행 획은 지우기 후보 박스에 들어오더라도 실제 경로에 닿지 않으면 유지되도록 회귀 테스트를 추가했습니다.

## 연필 브러시 최적화

- 연필 렌더링에서 저장된 원본 획은 유지하면서, 화면 렌더 단계에서만 너무 가까운 점을 줄입니다.
- 기존의 과도한 입자 레이어 반복을 중심 획, 제한된 질감 레이어, 상한이 있는 입자 효과로 조정해 확대 상태의 긴 연필 획 렉을 줄였습니다.
